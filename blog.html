What is DOM? What is the purpose of DOM?

Document Object Model simply DOM is programming interface for web documents like html and xml. When a webpage loaded, it create a document object model. It's design look likes a tree. It allow us to create, change or remove elements or styles from the document. Its purpose to make webpage more interactive. When we need to change from the html and xml file we need to work with DOM.

How we will select elements using DOM?

Easiest way to select an elements from html using DOM is getElementById selector. There are 5 ways to select elements using DOM.
1. getElementsByTagName() 2. getElementsByClassName() 3. getElementById() 4. querySelector() 5. querySelectorAll() We can select element by using their tag name by getElementsByTagName. Also class name by getElementsByClassName. we can also select element by querySelector. This will give the first element that matches with CSS selector. And querySelectorAll will give all matches elements with CSS selector. The basic syntax of writing them has been:
document.getElementsById('id');

What is event bubble?

Event bubble is the concept of DOM(Document Object Model). There are many events in DOM and html elements receive them. Event bubble one of them. Event bubble happen when a html element receive an event like click, mouseover, scroll, blur and many more. And event bubble ups to its parent element until it gets the root element that is document. If a button clicked that means an event happens with the button elements. And this event will go up to its parent element. it will propagate until it find its root element.

What is callback function? Why we will use it?

When a function takes a function as argument its called callback function. In short, it is a function that is executed after another function is executed. We know the javascript asynchronous behavior. When a function or a statement takes more time then it's go to the next statement without any waiting and execute the next statement. So that we cannot get the data in order that we want to get. For solve this problem callback function has been used.
